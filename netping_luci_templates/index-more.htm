Универсальные шаблоны ресурсов В информационной системе управления и монитоинга устройствами NetPing используются шаблоны. Шаблоны нужны для быстрого управления конфигурацией различных ресурсов. Ресурсом называется всякий объект управления, представленный в информационной системе. Примеры ресурсов: “датчик”, “реле”, “триггер”, “уведомление” и др. У всякого ресурса есть свои, присущие ему свойства (или атрибуты). Поэтому в шаблоне можно создавать произвольное число атрибутов данного ресурса. Такие атрибуты в шаблоне называются “Переменные шаблона”. Переменная шаблона имеет предопределённый заранее тип (или формат входных данных): строка, число, IP-адрес, и т.п. Это позволяет при создании ресурса по шаблону, избегать ошибок пользователя. Для удобства использования, шаблоны классифицированы по категориям.
