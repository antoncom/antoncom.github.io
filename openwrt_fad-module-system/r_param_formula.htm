Переключить если не в сети Правило создано по шаблону. В шаблоне предусмотрены стандартные переменные: name (имя правила), action (действие). Переменная action содержит вызов команды, выполняемой в зависимости от выполнения условия, указанного в модификаторе “Логическая функция”. Вы можете добавить вспомогательные параметры, например param и т.п.
